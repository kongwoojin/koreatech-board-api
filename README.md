## koreatech-board-api
비공식 한국기술교육대학교 공지사항, 게시판 클라이언트 REST API

## 사용된 기술 스택
* [Go](https://go.dev/)
* [EdgeDB](https://www.edgedb.com/)

## 사용된 라이브러리
* [echo](https://echo.labstack.com/)
* [edgedb-go](github.com/edgedb/edgedb-go)
